Over the last few months, I've been working on a consulting project for the Brooklyn Historical Society, whose catablog, Emma, runs on WordPress. Because of a large number of grant projects in which the BHS staff are using Archivists' Toolkit to add lots of resource records, they needed a way to add individual as well as sets of collection descriptions to WordPress using an easy procedure.

The solution I created for them was the BHS MARCXML Importer plugin. The plugin allows you to import data from XML files containing MARCXML records or Zip files containing sets of such XML files and create posts with that data. Currently, the mappings are mostly fixed, and tailored for MARCXML records that describe archival material.  The plugin requires Dan Scott's fabulous File_MARC PEAR module for parsing the records. Please feel free to test it and give me any feedback as appropriate. There is some detailed documentation available in addition to the readme file. I am also hosting the code using Github, so feel free to dig into the code there.

This tool was created as part of the project, "Uncovering the Secrets of Brooklyn's 19th Century Past: Creation to Consolidation," funded by the Council on Library and Information Resources, with additional support from The Gladys Krieble Delmas Foundation.
